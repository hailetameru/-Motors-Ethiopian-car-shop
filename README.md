# 🚗 Habesha Motors

Welcome to **Habesha Motors** – an elegant, modern, and fully responsive web application for Ethiopian car shoppers and enthusiasts.

## What is Habesha Motors?

Habesha Motors makes browsing, discovering, and exploring the latest cars in Ethiopia a breeze. Built with love using HTML, CSS, and JavaScript, it’s fast, beautiful, and works on any device.

## Features

- 🏎️ Browse a curated selection of cars available in Ethiopia
- 📱 Fully responsive – looks great on desktop, tablet, and mobile
- 🔍 Simple and intuitive navigation
- 🤩 Modern and attractive interface

## Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/hailetameru/-Motors-Ethiopian-car-shop.git
   cd -Motors-Ethiopian-car-shop
   ```

2. **Open `index.html` in your browser.**

No frameworks or build steps needed – just pure HTML, CSS, and JavaScript.
